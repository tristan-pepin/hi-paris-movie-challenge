# Backend

API FastAPI servant les données films.

## Prérequis

- Python 3.11+
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

## Installation

```bash
uv sync
cp .env.example .env
```

## Lancer

```bash
uv run uvicorn main:app --reload
```

API disponible sur `http://localhost:8000`.
